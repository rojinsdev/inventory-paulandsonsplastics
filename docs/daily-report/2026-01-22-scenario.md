# The Master Test Script: "From Day Zero to Payday"

This test scenario simulates a full lifecycle using the system's strict two-role structure: **Admin** and **Production Manager**.

## 🎭 The Roles

1.  **Admin** (Web Portal):
    *   Responsibilities: System Setup, User Management, Sales, Analytics, Finance.
    *   *You act as this role.*
2.  **Production Manager** (Mobile App & Web Log):
    *   Responsibilities: Daily Production Logging, Floor Inventory (Packing/Bundling), Raw Materials.
    *   *Test Persona*: "Ramesh Ops".

---

## 🏗️ Act 1: The Foundation (Admin)

**Goal**: Setup the organization and onboard your Production Manager.

### 1.1 Create the Admin User (Day 0)
*   **Prerequisite**: Login as the initial Admin user.

### 1.2 Onboard Production Manager
*   **Action**: Admin Portal -> Users -> **Create New User**.
*   **Entry**:
    *   Name: "Ramesh Ops"
    *   Email: `ramesh@paulandsons.com`
    *   Role: `production_manager`
    *   Password: `password123`
*   **Verify**: "Ramesh" can now login to the Mobile App.

## 🏭 Act 2: Organizing the Factory (Admin)

**Goal**: Define *what* you make and *how* you make it.

### 2.1 Define Machines
*   **Action**: Admin Portal -> Machines -> Create New.
*   **Entry**:
    *   Name: "**Extruder A1**"
    *   Type: `extruder`
    *   Running Cost: `₹7500` / day.

### 2.2 Define Products
*   **Action**: Admin Portal -> Products -> Create New.
*   **Entry**:
    *   Name: "**Super-Bottle 1L**"
    *   Weight: `25g`
    *   Color: `White`
    *   Packing: `100` per Packet, `50` Packets per Bundle.
    *   Selling Price: `₹0.80` (Price per piece, so 5000 units = ₹4000).

### 2.3 Map Machine to Product
*   **Action**: Admin Portal -> Machine Mapping.
*   **Logic**: Link "Extruder A1" to "Super-Bottle 1L".
*   **Setting**: Ideal Cycle Time = `6.0` seconds.

## ⚙️ Act 3: Production Day (Production Manager)

**Goal**: Ramesh runs the floor and ensures raw materials are ready.

*   **Login**: As `ramesh@paulandsons.com`.
*   **Action**: Inventory -> Raw Materials.
*   **Task**: Add Stock.
    *   Material: "Virgin HDPE Granules".
    *   Adjustment: `+5000 kg` (Opening Stock).

### 3.2 The Production Run (Production Entry Screen)
*   **Action**: Production Log -> **Add Entry**.
*   **Entry Details**:
    *   **Machine**: `Extruder A1`
    *   **Product**: `Super-Bottle 1L (White)`
    *   **Shift**: `1` (Day)
    *   **Total Produced**: `10,000` units.
    *   **Damaged Count**: `8` units.
    *   **Actual Cycle Time**: `6.2` seconds (From machine display).
    *   **Actual Weight per unit**: `25.2` grams (Measured weight).
    *   **Downtime Reason**: `N/A` (Leave empty if < 30 mins).
*   **Verification**: System auto-deducts ~252kg of Granules and flags as 'Cost Recovered'.

## 📦 Act 4: The Floor Operations (Production Manager)

### 4.2 Bundling (Packets -> Sacks)
