# Company Logo Instructions

## 📁 What to Place Here

Add your Paul & Sons Plastics company logo file to this folder:

1. **Logo File**: Save your logo as `logo.png`
   - Recommended size: **512x512px** or **1024x1024px**
   - Format: PNG (with transparent background preferred)
   - This will be used on the login screen and throughout the app

2. **Optional - White Logo**: If you have a white version for dark backgrounds:
   - Save as `logo_white.png`
   - Same size recommendations as above
   - Useful for colored AppBars or dark UI elements

## 📝 File Naming

- **logo.png** - Main company logo (REQUIRED)
- **logo_white.png** - White variant (optional)

## 💡 Tips

- Keep the file size under 500KB for best app performance
- Use a transparent background (PNG format) for versatility
- Square aspect ratio works best for mobile displays
- High resolution ensures crisp display on all devices

## 🔧 After Adding Logo

After placing your logo file here, the app will automatically use it in the redesigned screens!
